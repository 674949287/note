从输入URL到页面加载，简单来说分为以下几个过程：

1.输入URL  //后端发送到DNS服务器
2.DNS解析  // 首先会找本地浏览器的缓存、系统缓存、路由缓存，如果有缓存直接返回缓存内容，否则根据解析到的IP地址和端口向浏览器发送一个 HTTP请求
3.TCP连接  //与web服务器建立TCP连接（3次握手协议）1.客户端向服务端sayhi 2. 服务端向客户端sayhi 3.客户端收到提出聊天  进入互相交流
4.发送HTTP请求  // 浏览器向web服务器发送HTTP请求
5.服务器处理请求并返回HTTP报文 // 浏览器下载web服务器返回的数据
6.解析渲染页面 // 通过浏览器的渲染引擎=> 1.解析html源文件，构建dom树 2.解析CSS,形成CSS对象模型 3.将CSS与DOM合并，构建渲染树（renderingtree）4.布局和绘制
7.连接结束 // 4次挥手过程 => 1.客户端向服务端发起断开请求 2. 服务端接收，准备关闭连接 3. 服务端准备完毕，告诉客户端我关闭了 4.客户端收到，也关闭
